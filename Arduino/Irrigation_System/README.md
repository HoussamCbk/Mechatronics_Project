# **Irrigation System**
## This project is about watering plant when it becomes dry using ; 
+ Soil sensor
+ Soil sensor module
+ Relay 5v
+ Arduino Uno
+ Two 9v Batteries to power the pump and the arduino
+ Pump
+ Wires
